# 操作数据库cmd常用命令
## 非javascript 命令
1. help `mongodb支持的命令`  
2. show dbs (show databases) `查看所有数据库`  
3. use databsesname  `使用(切换)数据库`  
4. db  `查看当前的库名` 
5. db.hlep `数据库级别帮助命令`
6. show collections (show tables) `查看当前库下所有的集合`
7. db.collectionname.help  `查看表级别的帮助命令` 
8. db.shutdownServer()  `关闭mongod服务`
